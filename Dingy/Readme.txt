Nederlandse politieboot
=========================
Installatie:
Installeer eerst de boot hier: <a href='https://www.gta5-mods.com/vehicles/veteran-mods-fire-pack#description_tab">https://www.gta5-mods.com/vehicles/veteran-mods-fire-pack</a>.
Nadat je dat hebt gedaan, upload de .ytd bestanden in het zip bestand wat je hebt gedownload op dezelfde plek als het model.
Het .xml bestand moet in <i>ELS/pack_default</i>
_________________________

Dutch police boat
=========================
Installation:
First install the boat: <a href='https://www.gta5-mods.com/vehicles/veteran-mods-fire-pack#description_tab">https://www.gta5-mods.com/vehicles/veteran-mods-fire-pack</a>.
After that, upload the .ytd files in the zip file you just downloaded at the same place as the model.
Upload the .xml file in <i>ELS/pack_default</i>
_________________________
